## Avukat görüşmesinden önce doldurun

Aşağıdaki placeholder'ları gerçek bilgilerle değiştirin:

- `[LEGAL ENTITY NAME]` — LLC mi, sole prop mu?
- `[REGISTERED AGENT ADDRESS]` — Orange County tam adres
- `[DMCA AGENT NAME]` — DMCA Copyright Office kaydı için
- `[EIN / STATE OF FORMATION]` — varsa

## Avukat onayından sonra yapılacaklar

1. Terms sayfasına `06-Terms-Amendments-DRAFT.md` maddelerini ekleyin
2. `/legal/dmca` ve `/legal/community-guidelines` sayfalarını yayınlayın
3. DMCA Designated Agent'ı **U.S. Copyright Office**'e kaydedin
4. Mobil uygulamaya `04-In-App-Legal-Copy.md` metinlerini ekleyin
5. Sign-up ve PIN submit akışlarına checkbox/onay ekleyin
